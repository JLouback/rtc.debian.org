

This is a web site, not a package.

It is normally published to http://rtc.debian.org

Debian Developers with authorized access to the web server
can publish from this repository with the enclosed deb-web-publish
script.

This repository is a snapshot of a live web site rather than a pure
source repository.  Minified JavaScript artifacts have all been
taken directly from the respective Debian packages.  The source code
of these artifacts can be found in the source packages.
